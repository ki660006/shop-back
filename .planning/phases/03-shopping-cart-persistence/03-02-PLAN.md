        - Find or create cart.
        - Validate product existence and stock (from `ProductRepository`).
        - Enforce limits: Max 99 per item, Max 50 unique items in cart.
        - Add or increment quantity.
    - `updateQuantity(CartIdentifier identifier, UUID itemId, int quantity)`: Update specific cart item quantity with stock check.
    - `removeItem(CartIdentifier identifier, UUID itemId)`: Remove specific item.
  </action>

  <action>
    Create REST controller and DTOs:
    - `GET /api/cart`: Returns current cart based on authenticated user or `guest_cart_id` from header/param.
    - `POST /api/cart`: Adds item to cart.
    - `PUT /api/cart/items/{itemId}`: Updates item quantity.
    - `DELETE /api/cart/items/{itemId}`: Removes item.
    
    Response DTOs should include:
    - `CartResponse`: list of items, subtotal.
    - `CartItemResponse`: productId, name, price, quantity, mainImageUrl.
  </action>
