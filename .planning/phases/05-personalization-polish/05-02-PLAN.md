    3. `getRecommendations(userId, limit)`: Implement the Hybrid Querydsl logic.

       - If not enough recommendations exist, fall back to global bestsellers.
       
    Create `RecommendationController` to expose:
    - `POST /api/products/{id}/view`
    - `GET /api/products/recent`
    - `GET /api/recommendations`
  </action>
